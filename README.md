# 도로터널 방재·환기시설 검토시스템 2026

국토교통부예규 제461호(시행 2026.04.30) 기준 도로터널 방재·환기시설 검토 보조 웹앱입니다.

## Version

- v1.2

## 포함 기능

- 일반 도로터널 / 소형차 전용터널 입력 전환
- 방음터널 추가 위험인자 입력
- 연장등급 산정
- 위험도지수(X) 산정
- 위험도 기준 방재등급 산정
- 최종 적용 방재등급 산정
- 계산근거 표시
- 방재시설 설치기준 요약표
- URL 공유
- JSON 저장
- 인쇄/PDF

## 주의

본 도구는 실무 검토를 돕는 보조도구입니다. 최종 판단은 공식 지침 원문과 관리기관 검토 결과를 우선합니다.


## RC2 변경사항

- 상단 탭 추가: 계산기 / 평가기준 / 등급별 방재시설
- 2026 평가기준 요약표 추가
- 등급별 방재시설 요약표 추가


## v1.0.1 수정사항

- 주행거리계(L×AADT) 계산 시 터널연장을 m에서 km로 변환하도록 수정
- 대형차 주행거리계 및 소형화물 주행거리계도 동일하게 km 기준으로 수정


## v1.1.1 수정사항

- PDF 원문 기준에 맞춰 입·출구 표고차 평가기준을 10m 단위 기준으로 수정
- 일반 도로터널: 10m / 20m / 30m 기준 반영
- 소형차 전용터널: 10m / 20m / 30m / 40m / 50m 기준 반영
- 평가기준표와 계산엔진을 동일 기준으로 업데이트


## v1.2 수정사항

- UI 제목을 “도로터널 방재·환기시설 검토시스템”으로 변경
- 평가기준표의 터널특성 “입·출구 표고차(m)” 범위를 PDF 원문 기준에 맞게 10m 단위로 수정
- 일반 도로터널 입·출구 표고차 계산기준: 10m / 20m / 30m 기준 반영
- 소형차 전용터널 입·출구 표고차 계산기준: 10m / 20m / 30m / 40m / 50m 기준 반영
- `css`, `data`, `js` 폴더 구조로 정리
