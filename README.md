# 도로터널 방재·환기시설 등급 산정기 2026

국토교통부예규 제461호(시행 2026.04.30) 기준의 도로터널 방재·환기시설 등급 산정기입니다.

## 배포
GitHub Pages: `main` 브랜치 `/root` 기준으로 배포합니다.

## 파일 구성
- `index.html`: 메인 화면
- `css/style.css`: 화면 스타일
- `js/calculator.js`: 계산 로직
- `js/app.js`: UI 동작
- `data/safety2026.js`: 기준 메타데이터

## 주의
현재 버전은 1차 배포용 초안입니다. 표 1.2.3~1.2.5의 시설별 설치기준 표시는 후속 버전에서 추가됩니다.
