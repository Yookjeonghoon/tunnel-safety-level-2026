# 도로터널 방재·환기시설 등급 산정기 2026

국토교통부예규 제461호(시행 2026.04.30) 기준 도로터널 방재·환기시설 등급 산정 보조 웹앱입니다.

## Version

- v1.0 RC21

## 포함 기능

- 일반 도로터널 / 소형차 전용터널 입력 전환
- 방음터널 추가 위험인자 입력
- 연장등급 산정
- 위험도지수(X) 산정
- 위험도 기준 방재등급 산정
- 최종 적용 방재등급 산정
- 계산근거 표시
- 방재시설 설치기준 요약표
- URL 공유
- JSON 저장
- 인쇄/PDF

## 주의

본 도구는 실무 검토를 돕는 보조도구입니다. 최종 판단은 공식 지침 원문과 관리기관 검토 결과를 우선합니다.


## RC2 변경사항

- 상단 탭 추가: 계산기 / 평가기준 / 등급별 방재시설
- 2026 평가기준 요약표 추가
- 등급별 방재시설 요약표 추가
