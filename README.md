# 도로터널 방재·환기시설 등급 산정기 2026

국토교통부예규 제461호(시행 2026.04.30) 기준 도로터널 방재·환기시설 등급 산정 보조 웹앱입니다.

## Version

- v0.5

## v0.5 변경사항

- 일반 도로터널 위험도지수 계산 보강
- 소형차 전용터널 위험도지수 계산 보강
- 방음터널 추가 위험인자 입력 및 점수 반영
- 최종 적용 방재등급 산정 로직 유지
- 방재시설 설치기준 요약표 추가
- JSON 저장 버전 정보 갱신

## 포함 기능

- 연장등급 산정
- 위험도지수(X) 산정
- 위험도 기준 방재등급 산정
- 최종 적용 방재등급 산정
- 계산근거 표시
- URL 공유
- JSON 저장
- 인쇄/PDF

## 주의

본 도구는 실무 검토를 돕는 보조도구입니다. 최종 판단은 공식 지침 원문과 관리기관 검토 결과를 우선합니다.
